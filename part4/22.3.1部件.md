# 22.3.1部件

ID

用易于理解的惟一标识符命名测试包。

包

測试链Q12028(包含第一季度运行的所有測试用例。当报告按季度编制时，这是可行的。每个季度都会进行审批，并运行特殊的批处理程序。

内容

测试包是一种分组测试用例的方法，您必须或应该在特定的时间点、同时或以特定的顺序运行测试用例。将所有测试用例设置为在同一天运行在相同的测试包中有一个明显的优势这样管理和执行测试就变得更容易了。测试包可以按日期分开。你必须自己決定什么适合你自己的项目。

为了验证项目中的业务流程，我们创建了五个测试包。East测试包包含了所有业务流程在特定季度运行的所有测试用例。
